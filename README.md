#  FamilyHub - Семейный органайзер

Полнофункциональное веб-приложение для управления семейными задачами, отслеживания прогресса и координации между членами семьи.

##  Особенности

### Управление задачами
-  Создание и назначение задач членам семьи
-  Категории задач (Уборка, Готовка, Покупки, Дети, Питомцы, Финансы)
-  Периоды выполнения (День, Неделя, Месяц)
-  Двухэтапное подтверждение (ребёнок выполняет → родитель подтверждает)
-  Удаление и редактирование задач

### Система ролей
-  **Родитель** - полный доступ, подтверждение задач, управление семьёй
-  **Ребёнок** - выполнение своих задач, ограниченный доступ

### Геймификация
-  Система очков (10 очков за подтверждённую задачу)
-  Недельные цели:
  - Выполнить задачи (0-50)
  - Семейные очки (0-1000)
  - Активные дни подряд (0-7)
-  Визуализация прогресса в реальном времени

### Семейный профиль
-  Управление членами семьи
-  Загрузка собственных аватаров (JPG, PNG, GIF, WebP до 5MB)
-  Персонализация цветов
-  Отображение очков и статистики

### Отчёты (только для родителей)
-  Генерация отчётов о семейной активности
-  Экспорт в TXT и PDF форматы
-  История отчётов с детальной статистикой

##  Функционал

### 1. Аутентификация
- **Регистрация** - создание семейного аккаунта
  - Название семьи
  - Имя родителя
  - Email и пароль (мин. 6 символов)
- **Вход** - авторизация по email/паролю
- **Регистрация детей** - привязка к родительскому аккаунту

### 2. Дашборд
- **Метрики реального времени:**
  - Всего задач
  - Выполнено задач
  - В процессе
  - Всего очков
- **Недельные цели** с прогресс-барами
- **Активность семьи** - аватары членов с очками
- **Календарь** - визуальное отображение месяца

### 3. Центр задач
- **Создание задач:**
  - Название
  - Категория (6 типов)
  - Назначение члену семьи
  - Период (день/неделя/месяц)
- **Фильтрация:** Все / Сегодня / На неделе
- **Статусы:**
  - ⏳ Ожидает выполнения
  - ✅ Выполнено (ожидает подтверждения)
  - ✔️ Подтверждено (+ очки)

### 4. Управление семьёй
- **Просмотр** всех членов с карточками
- **Добавление** новых членов
- **Редактирование:**
  - Имя, роль
  - Аватар (загрузка или случайный)
  - Цвет очков
  - Количество очков
- **Удаление** членов семьи

### 5. Отчёты (Parent only)
- **Генерация отчётов:**
  - Дата создания
  - Статистика (задачи, очки, члены семьи)
  - Топ исполнителей
- **Экспорт:**
  - 📄 Формат TXT
  - 📕 Формат PDF
- **История** сохранённых отчётов

### 6. Настройки
- **Уведомления** - вкл/выкл
- **Тема** - светлая/тёмная
- **Информация о семье**
- **Выход** из аккаунта
